(`reconstruction/dossiers/`, passes 1–4) per the unit-truth-spec's
research → anchor → **place** sequencing. The founding directive this wave
serves: *"most of the blocks aren't involved in the action … put them at
different points at different times"* — under the counter-rule that
attested stillness stays still and nothing is ever invented. Every change
below carries its dossier citation in the battle file itself.

Timing frame unchanged: t=0 = startTime 46800 = nominal 13:00 (the chain
reads it 13:07; `shippedSkewMinutes: -7` stays a label note, per ED-24 —
nothing was re-timed). All additions sit inside the existing window
(t=0..10800); window widening is a future wave.
